## Installation

1. Clone the repository

```bash
gh repo clone PiyushThakur1/url-shortner
cd URL-Shortener
```

2. Install dependencies

```bash
npm install
```

3. Start the Server

```
npm start
```

Alternatively, you can run it directly:

```
node app.js
```

4. Start frontend Server

```
cd public
npx serve
```
